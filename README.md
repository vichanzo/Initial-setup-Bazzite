## Application of the WhiteSur theme (from https://github.com/Imasu/Guide-Installation-Bazzite)

Create a distrobox for the theme compilation with a separate home folder

````
distrobox create -i "fedora:latest" -n "sys-deskcustom" -H "$HOME/.containers-home/sys-deskcustom"

````
In the distrobox, install gitand all the dependencies requested: sudo dnf install git <packages>
In the distrobox, download the theme and install it with the commands

```
cd
git clone https://github.com/vinceliuice/WhiteSur-gtk-theme.git --depth=1
cd WhiteSur-gtk-theme
./install.sh -l -HD
./tweaks.sh -F
````

From the host system console, copy the contents of the following directories to their host system counterpart
````
cp -r $HOME/.containers-home/sys-deskcustom/.config/gtk-4.0 $HOME/.config/
cp -r $HOME/.containers-home/sys-deskcustom/.themes/*  $HOME/.local/share/themes
````
From the host system console, change the two link files that point to the container directory
````
ln -s -f $HOME/.config/gtk-4.0/gtk-Dark.css $HOME/.config/gtk-4.0/gtk.css
ln -s -f $HOME/.config/gtk-4.0/gtk-Dark.css $HOME/.config/gtk-4.0/gtk-dark.css
````
Install the Flatpaks generated by the installation
The installation script generated flatpak files in the directory $HOME/.containers-home/sys-deskcustom/.cache/pakitheme
From the host system console, go to this directory and for each subdirectory install the flatpak in system mode:
````
[skip] cd $HOME/.containers-home/sys-deskcustom/.cache/pakitheme
[skip] flatpak --system install WhiteSur-Dark/org.gtk.Gtk3theme.WhiteSur-Dark-x86_64.flatpak
[skip] flatpak --system install WhiteSur-Dark-solid/org.gtk.Gtk3theme.WhiteSur-Dark-solid-x86_64.flatpak
[skip] flatpak --system install WhiteSur-Light/org.gtk.Gtk3theme.WhiteSur-Light-x86_64.flatpak
````
Set up Flatpak permissions from FlatSeal
Add to the section Filesystemfor all applications the following permissions
````
/var/home/$user name$/.local/share/icons/*:ro
/var/home/$user name$/.local/share/themes/*:ro
xdg-config/gtk-4.0:ro
xdg-config/gtk-3.0:ro
````
